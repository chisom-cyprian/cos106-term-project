# Chisom Cyprian - Software Engineering Portfolio

Welcome to my professional engineering portfolio. This repository contains the source code for my responsive, high-contrast dashboard web interface showcasing my software architecture journey, core academic foundations, and autonomous automation workflows. 

## 🚀 Live Demo
🔗 **Live Production Site:** https://chisom-cyprian.vercel.app/

---

##  Architecture & Core Stack
The interface is built using clean, accessible, native web technologies focused on semantic performance, responsive design layers, and fluid scaling systems:

* **Markup:** Semantic HTML5 Structure
* **Styling:** Modular CSS3 Custom Properties (Variables) & Responsive Media Queries
* **Interactivity:** Lightweight Vanilla JavaScript (ES6+)
* **Deployment & CI/CD:** GitHub Pages Compilation Pipeline

---

##  Featured Showcases & Academic Foundations
The platform organizes my technical projects, core tools, and focus areas:

* **Beyond the Screen Dashboard:** Implements custom media rendering container wrappers utilizing `object-fit: cover` matrix rules to smoothly scale portrait and landscape project tracking videos.
* **JPMC Transaction Processing:** An enterprise-grade simulation system built utilizing Java and Spring Boot for backend transaction processing infrastructure.
* **CS50 AI Portfolio:** A collection of twelve technical implementations covering search algorithms, knowledge representation models, optimization problems, and machine learning neural networks using Python and `scikit-learn`.
* **Integrated Task Tracker:** A clean component module designed to map real-time sprint goals and course milestones cleanly.

---

##  Theme: High-Contrast Light Studio
The design system drops heavy dark modes in favor of a modern, highly legible developer documentation layout inspired by top-tier modern tech ecosystems (Stripe, Vercel, GitHub Light):
* **Background:** Pure white canvas (`#ffffff`) mixed with subtle slate-tinted raised panels.
* **Typography:** Clean font pairings with sans-serif structural headings and high-contrast monospace code metadata stamps.
* **Signature Accent:** Electric Brave Orange (`#ff7636`) for interactive indicator states, badges, and focus borders.
